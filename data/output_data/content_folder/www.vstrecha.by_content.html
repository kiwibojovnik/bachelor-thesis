              Сообщить результат


              Мы просим всех, кто прошёл экспресс-тестирование на ВИЧ с помощью нашего набора для самотестирования, сообщить нам о своих результатах. Ты сообщаешь анонимно только свой регион проживания и индивидуальный код, который создал самостоятельно при заказе экспресс-теста. Наш сотрудник вписал его в бланк, который ты получил в наборе для самотестирования.

              Внимательно посмотри на свой экспресс-тест, сравни результат с картинкой. Интенсивность цвета полоски на результат теста не влияет!

              ВИЧ-Отрицательный результат

              ОДНА ЛИНИЯ рядом с «C» и НЕТ линии рядом с «T

              ВИЧ-Положительный результат

              ДВЕ ЦЕЛЬНЫЕ ЛИНИИ, даже если одна из них слабовыражена. Возможен
              положительный результат. Необходимо провести дополнительный тест.

              Результат недействительный

              НЕТ ЛИНИИ рядом с «C» (даже тогда, когда есть линия рядом с «Т»),
              или красный фон, мешающий прочитать тест. Тест не сработал,
              необходимо пройти повторный тест.

              Если ты хочешь, чтобы наш сотрудник сам с тобой связался, пожалуйста, оставь свои контакты на выбор:


              Возможно, у тебя есть вопросы по результатам теста или твоим
